## Customization

### Update Personal Information

Edit the following files to customize with your information:

- `components/Hero.tsx` - Update name, title, and social links
- `components/About.tsx` - Update about section content
- `components/Skills.tsx` - Add/modify your skills
- `components/Projects.tsx` - Add your projects with images and links
- `components/Contact.tsx` - Update contact information

### Styling

- `app/globals.css` - Global styles and custom animations
- `tailwind.config.ts` - Tailwind configuration and custom animations
